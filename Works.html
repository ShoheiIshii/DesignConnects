ポートフォリオ

担当範囲

デザイン・コーディング・イラスト・ライティング

制作期間

1週間

使用ツール

Figma / Illustrator / CLIP STUDIO PAINT / Photoshop HTML / CSS / jQuery

自身のポートフォリオデザインを制作しました。（Figma）
コンセプトやブランドイメージを視覚化したい、ブランドイメージを定めたいクライアント様をターゲットに仕事をいただきたく内容を決定しました。また、自身と同じようにコミュニケーションを大事にする方々の目に留まれば良いなと思い、テーマを決めました。

制作のポイント

  • 私が「どんな人のため」に「何ができるのか」が伝わるように訴求内容と構成を設計した。
  • サイト全体を緑色にすることで、その色彩効果で、ストレスを感じにくく少しでも読み進めやすいよう意識した。
